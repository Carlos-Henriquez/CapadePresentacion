Este programa, "CapadePresentacion", fue realizado en C# mediante la utilización del IDE Visual Studio 2022.

Es una calculadora con interfaz grafico por en el cual al ingresar los numeros en ambas casillas, estos se multiplicaran y el resultado se mostrara en el area señalada. Dentro del programa se pude utilizar la tecla tab para navegar entre los dos cuadros y el boton que realiza la operación de multiplicar.
